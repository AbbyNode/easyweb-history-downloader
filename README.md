# EasyWeb History Downloader

This is a userscript for downloading your transaction history from TD EasyWeb (TD Canada Trust online banking).

## Features
- Downloads your account transaction history as a CSV file.
- Works directly from the EasyWeb website.

## Installation
1. **Install a userscript manager**
   - [Tampermonkey](https://www.tampermonkey.net/) (recommended)
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [Greasemonkey](https://www.greasespot.net/)

2. **Install the script**
   - Download or copy the contents of `easyweb-history-downloader.user.js` from this repository.
   - In your userscript manager, create a new script and paste the contents, or use the "Install from file" option if available.

3. **Go to TD EasyWeb**
   - Log in to your TD EasyWeb account.
   - Navigate to your account transaction history page.

4. **Use the script**
   - A new button or menu item should appear (usually labeled "Download History" or similar).
   - Click the button to download your transaction history as a CSV file.
